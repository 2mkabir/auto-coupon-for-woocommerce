Automatically add the coupon to the customer\'s cart or the admin order page if the restrictions are met.

== Description ==
Allow a coupon to be applied automatically by checking \"Auto apply\" in edit coupon page.
Automatically add the coupon to the customer\'s cart or the admin order page if the restrictions are met.
Please enter a description when you check \"Auto apply\" box, the description will be shown in the customer\'s cart if the coupon is applied.

== Installation ==
1. Upload the plugin in the `/wp-content/plugins/` directory, or automatically install it through the \'New Plugin\' menu in WordPress
2. Activate the plugin through the \'Plugins\' menu in WordPress
